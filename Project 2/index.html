Project Summary






The purpose of this project was to analyze if there is a relationship between certain audio features of a song and it's popularity.

To accomplish this analysis, we first pulled the data from the Spotify Web API. We called the API using multiple loops to find 2000 tracks released in 2019.Using the track_id we were able to obtain some audio features such as: danceability, energy, liveness, loudness, and the music genre that each artist belongs to. We determined the characteristics shared by the most popular songs available on Spotify.

    Questions:
  1. Do faster, more upbeat songs tend to be more or less popular than slower songs?
  2. What other audio features relate to the popularity of a song?
  3. Which musical genres have the highest popularity ratings?
  4. Can these findings be used to predict the popularity of a new song?

Data Sets Used:
Data was obtained by calling and looping through the Spotify Web API. There are three separate loops. The first loop to get lists of track names, artist names and unique track ids. The second to use the track id to find the audio features for each track. The third to use the artist name to determine the genre.
